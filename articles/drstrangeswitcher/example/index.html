  • Dmitri? Dmitri! You can switch to the default style now
  • Why don't you switch the styles, eh?!

This is a little example of Dr. Strangeswitcher, an article written by Patrick Griffiths that explains how you can construct a style-switcher with nothing more than plain old HTML and CSS.

The whole page becomes one big page anchor, and alternative styles are applied by targeting that "page anchor" when it is linked to using CSS 3's :target pseudo class.

Unfortunately, you will require a decent, modern browser, such as Firefox or Safari to see this basic example at work...

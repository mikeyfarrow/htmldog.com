CSS Value: Angle

The turn between two lines radiating from a single point, most commonly measured in degrees.

Angles can be used with CSS transformations and gradients.

When an angle is used to suggest a direction (such as with linear-gradient), 0deg, or equivalent, points straight upwards, 90deg points right, 180deg points down, and 270deg (or -90deg) points left.

A static angle can be represented in many ways; 0deg will have the same effect as 360deg or 3600deg (each of these turns reaches the same point on the circle). If angles are used in conjunction with animations, such as transitions, however, every turn of the circle will be honoured; 3600deg would make ten turns, for example.

Unit Description Example
deg Degrees. There are 360 degrees in a circle. body { background: linear-gradient(20deg, orange, red); }
grad Gradians. There are 400 gradians in a circle. .photo { transform: rotate(50grad); }

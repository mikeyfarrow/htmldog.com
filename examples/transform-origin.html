Transform origin

Using the transform-origin CSS property to alter the point at which transformations are measured from. Using a rotated 2D transformation as an example.

transform: rotate(10deg);
transform-origin: 50% 50%;
/* default */
transform: rotate(10deg);
transform-origin: top;
transform: rotate(10deg);
transform-origin: right;
transform: rotate(10deg);
transform-origin: right top;
transform: rotate(10deg);
transform-origin: 25% 25%;
transform: rotate(10deg);
transform-origin: -20px 20px;
